# Interview Prep App

A Vue 3 interview prep UI that walks users through a mock interview session with typewriter-style animations and smooth page transitions.

## Tech Stack

- **Vue 3** (Composition API + `<script setup>`)
- **Vue Router** with lazy-loaded routes
- **TypeScript**
- **Vite**

## How It Works

1. **Setup** — Click "Generate Interview Scenarios" twice to walk through the two-step setup flow and enter the session.
2. **Welcome** — A typewriter-animated greeting introduces the session. Click "Yes, Continue" to begin.
3. **Questions** — Practice through interview questions with word-by-word animated answers, a progress bar, and smooth transitions between questions.
4. **Results** — Review your score and general advice.

```sh
npm run dev
```
